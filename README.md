# Ejecutar en Desarrollo

1.- Clonar Repositorio
2.- Ejecutar
```
yarn install

```
3.- Instalar nestCLI
````
npm o -g @nestjs/cli

```` 

4.- Levantar Base de Datos
````
docker-compose up -d
````


## Stack usado
* MongoDB
